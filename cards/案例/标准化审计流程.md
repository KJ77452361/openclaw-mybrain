# 标准化审计流程

> 类型：案例
> 创建：2026-06-27
> 标签：#复盘

## 摘要

使用 standard-auditor skill 对所有 Agent 工作区进行合规审计，确保 21/21 通过。

## 详情

- 审计项：AGENTS.md / SOUL.md / IDENTITY.md / USER.md / MEMORY.md 5 项必选
- 工具：grep + 脚本自动化批量检查
- 问题处理：立即修复，不积累技术债务
- 结果：21/21 agents 全部通过

## 经验教训

- 架构调整前先评估影响面，避免遗留问题
- 标准化工作要彻底，不能留半成品
- 执行和记录同步，避免事后补档
