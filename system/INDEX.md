## 定位

全局知识库（LYT 框架），位于 `~/.openclaw/workspace/MyBrain/`。
本文档是知识库的唯一入口和导航中心。

## 体系结构（对应 IATF16949 §7.5）

```
MyBrain/
├── system/                 ← 二阶文件（程序文件）
│   ├── INDEX.md            ← 本文件
│   ├── 导航.md             ← 快速导航
│   ├── 知识库标准.md       ← 知识库总标准（KB_STD_001）
│   ├── 卡片生命周期SOP.md  ← 卡片全流程（KB_SOP_002）
│   ├── 文件变更控制SOP.md  ← 变更管理（KB_SOP_003）
│   └── 受控文件清单.md     ← 受控文件台账（KB_REC_001）
│
├── cards/                 ← 三阶文件（作业文件）
│   ├── 概念/（6张）
│   ├── 方法/（4张）
│   └── 案例/（3张）
│
├── daily/                 ← 记录表单（四阶）
├── projects/              ← 项目文档
├── templates/            ← 模板库
├── calendar/             ← 日程记录
├── inbox/               ← 收集箱
└── external/            ← 外来文件
```
