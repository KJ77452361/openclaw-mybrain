# 外部工具集成规范

> 编码：QL_SOP_001_v1.0.0 | 版本：1.0.0 | 2026-06-27 | 状态：active
> 监管要求：IATF16949:2016 §7.5 成文信息控制
> 适用范围：所有向 OpenClaw 生态集成外部工具/服务的操作

## 1. 目的

规范外部工具（如 Codex++、Feishu、MiniMax API 等）的集成流程，确保：
- 外部工具接入经过安全评估
- 凭证（API Key、Token）不泄露
- 外部工具的使用有日志可追溯
- 外部工具的失效有降级方案

## 2. 适用范围

- OpenClaw 之外的任何服务（Feishu API、MiniMax API、Codex++ 等）
- SSH 远程调用（Windows cmd）
- Webhook 接收
- 文件系统级集成

## 3. 外部工具分类

| 类别 | 示例 | 风险等级 |
|------|------|---------|
| 消息通道 | Feishu | 高（数据外泄风险） |
| AI 能力 | MiniMax API | 中（成本风险） |
| 开发辅助 | Codex++ | 中（执行命令风险） |
| 文件系统 | SSH/SMB | 高（系统级访问） |
| Webhook | 外部回调 | 高（安全风险） |

## 4. 集成前评审

### 4.1 安全评估清单

| 评估项 | 标准 | 判定 |
|--------|------|------|
| 凭证安全性 | 凭证是否可 revocation | ✅/❌ |
| 数据流向 | 数据是否离开本地环境 | 明确 |
| 日志记录 | 集成是否可审计 | ✅/❌ |
| 故障处理 | 失败时是否有降级方案 | ✅/❌ |
| 成本控制 | 是否有用量上限/告警 | ✅/❌ |
| 依赖性 | 是否为单点依赖（无备用） | ✅/❌ |

### 4.2 凭证管理

所有外部工具凭证必须满足：

| 要求 | 说明 |
|------|------|
| 不明文存储 | 凭证不得出现在 .md 文件中 |
| 环境隔离 | 优先使用环境变量或配置文件 |
| 最小权限 | 凭证权限 = 最小所需权限 |
| 可 revocation | 凭证可随时撤销而不影响系统 |

**错误示例（禁止）：**
```markdown
API Key: sk-cp-abc123xyz
```

**正确示例：**
```markdown
API Key: sk-cp-***（已脱敏）
# 真实凭证存储在环境变量或 secrets 管理器
```

## 5. 集成实施流程

### Phase 1：评估

| 步骤 | 操作 | 产出 |
|------|------|------|
| 1.1 | 识别工具用途和必要性 | 用途说明 |
| 1.2 | 安全评估（§4.1） | 评估报告 |
| 1.3 | 成本估算 | 成本报告 |
| 1.4 | 确定凭证存储方案 | 存储方案 |

### Phase 2：设计

| 步骤 | 操作 |
|------|------|
| 2.1 | 设计调用接口（工具参数、返回值） |
| 2.2 | 设计错误处理和降级方案 |
| 2.3 | 设计日志记录方式 |
| 2.4 | 更新 `TOOLS.md` 记录工具配置 |

### Phase 3：实施

| 步骤 | 操作 |
|------|------|
| 3.1 | 配置凭证（环境变量或 secrets 管理器） |
| 3.2 | 编写调用代码/脚本 |
| 3.3 | 实现错误处理 |
| 3.4 | 实现日志记录 |
| 3.5 | 测试（正常 + 异常） |

### Phase 4：上线

| 步骤 | 操作 |
|------|------|
| 4.1 | 更新 `TOOLS.md`，新增工具记录 |
| 4.2 | 通知相关方（哪些 Agent 会使用） |
| 4.3 | 在 `daily/` 记录上线时间和凭证配置方式 |
| 4.4 | 设置监控告警（如用量超限） |

## 6. 日志记录要求

每次外部工具调用必须记录：

| 字段 | 说明 |
|------|------|
| 时间戳 | 调用时间（YYYY-MM-DD HH:MM） |
| 工具名称 | 被调用的工具 |
| 调用者 | 哪个 Agent/哪个流程调用 |
| 输入摘要 | 输入参数（脱敏后） |
| 输出摘要 | 输出结果（脱敏后） |
| 状态 | 成功/失败 |
| 耗时 | 执行时长（ms） |

## 7. 故障处理

| 故障类型 | 处理方式 |
|---------|---------|
| 工具返回错误 | 记录错误，按降级方案处理 |
| 凭证失效 | 触发 P1 告警，立即更换凭证 |
| 工具超时 | 增加超时限制，重试 ≤3 次 |
| 工具不可用 | 切换降级方案，记录事件 |

## 8. 已知工具配置

| 工具 | 凭证存储 | 调用方式 | 备注 |
|------|---------|---------|------|
| Feishu | 飞书应用管理后台 | OpenClaw 内置 Feishu 插件 | failure destination 待配置 |
| MiniMax | 环境变量 | OpenClaw 内置 | — |
| Codex++ | SSH 环境变量 | `ssh cmd /c codex_task.bat` | 结果存 `C:\Users\Administrator\codex_result.txt` |

## 9. 禁止事项

- **禁止**将真实 API Key / Token 明文写入任何 .md 文件
- **禁止**在群聊中发送凭证相关内容
- **禁止**在未经评审的情况下新增外部工具
- **禁止**在外部工具调用时不记录日志
- **禁止**使用无法 revocation 的凭证
